# HSNR-info
Honina is a infoserver with NLP,TTS capabilities (like Alexa)
This module uses api.ai for NLP and gtts for TTS.

#Dependencies
api.ai python api:
pip install apiai
text to speech (tts):
sudo pip install gTTS
lightweight mp3 player:
sudo apt-get install mpg321
JSON files:
pip install simplejson
